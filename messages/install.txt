SublimeLinter-contrib-rubycritic
-------------------------------
This linter plugin for SublimeLinter provides an interface to rubycritic.

** IMPORTANT! **

Before this plugin will activate, you *must*
follow the installation instructions here:

https://github.com/PedroCarmona/SublimeLinter-contrib-rubycritic

1. install rubycritic
    ```
        gem install rubycritic
        rbenv rehash
    ```
1. replace the path to rubycritic
    ```
        which rubycritic
    ```
